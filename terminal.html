Comandos basicos de la terminal CLI

pwd (print work directory): muestra la ruta donde se esta ejecutando la terminal de comandos

/ muestra la raiz de la ubicacion

GUI graphic user interface (explorador de archivos)

whoami: muestro el usuario

help: muestra todos los comandos

help pwd: muestra ayuda sobre el comando especifico y las banderas (-L -P -W). Tambien funciona, segun el entorno pwd --help o pwd -h

con las direccionales up ^ y v down recordamos el historial de comandos escritos

Para limpiar la terminal se usa clear

para listar los archivos dentro de una carpeta se usa ls, ls -l muestra mas informacion de los archivos, ls -a muestra tambien los archivos ocultos, ls -la muestra archivos detallados y ocultos

para cambiar de directorio se usa cd: chage directory, ej: cd otro-directorio

Atajos para el comando cd

se puede usar tabulador para autocompletar una palabra o extension dentro de una carpeta ej: si dentro del directorio hay un archivo css y nos vamos a mover ahi, se puede escribir cd c y tab, esto buscara los archivos que comiencen con c

se puede escribir cd y arrastrar el archivo hasta la terminal, y alli se mostrara la ruta exacta donde esta el archivo

cd ~ te regresa al home del usuario inicial

cd . te mueve al mismo directorio

cd .. lleva al directorio contenedor/padre, es como mover un nivel atras cd ../.. sube dos niveles

cd / mueve a la raiz del sistem, al disco duro, en bash cd c -> lleva al disco c cd d -> lleva al disco d cd - -> regresar a la carpeta anterior

para entrar en una carpeta que tiene espacios en blanco en su nombre se debe escribir su nombre entre comillas para evitar el error "to many arguments" asi -> cd "program files"

comando touch -> para crear un nuevo archivo ej: touch hola-mundo.txt echo "texto" > archivo.txt -> esto tambien crea y guarda una archivo

visualizar el contenido de un archivo se usa cat archivo.txt

crear una carpeta mkdir ej: mkdir assets

eliminar carpetas vacias rmdir ej: rmdir assets

rm es para eliminar archivos rm -r elimina directorios con sus archivos rm -rf borra forzosamente directorios y su contenido

mv: se usa para mover un archivo mv archivo.txt nombre de la carpeta una variante del comando mv permite renombra archivos ej: mv archivo.txt script.txt se deja asi si va a queda en la misma carpeta

copiar un archivo, se usa cp cp archivo.txt ../ copiar y renombrar cp file.txt ../ file1.txt copiar el contenido de una carpeta cp -r js css -> copia todo lo que este en js en css

para buscar archivos en la terminal find file find j* todo lo que empiece con j

para listar procesos activos se usa ps

para matar un proceso, se usa kill kill + id del proceso

code abriria un nuevo vsc, funciona con cualquier otro ide

crear alias sirve para crear claves para comandos muy largos alias rl="una ruta muy larga que no quiero escribir siempre" luego escribo alias para listar los alias creados escribo el alias rl

unalias elimina un alias creado previamente -> unalias nombre del alias al listar los alias ya no estara
