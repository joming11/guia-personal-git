Atajos de teclado vsc

    formas de generar estructuras con emmet

    html:5 o ! genera toda la estructura basica de un documento html

    nav>ul>li -> genera nodos hijos

    div+article+section -> genera nodos hermanos

    ul>li>p^li>blockquote -> para volver a un nodo superior

    nav>ul>li*5>a -> para multiplicar un elemento

    ul>li*3>p+span*2 -> esta expresion solo multiplica el span

    ul>li*3>(p+span)*2 -> esta expresion multiplica el p y el span

    nav#menu -> nav con id menu

    nav.menu -> nav con clase menu

    a.btn.btn-large.btn-white -> multiclases

    nav.social-media>a[target="_blank"]*3 -> para agregar un atributo

    nav.social-media>a[target="_blank"] [role="link"] -> para agregar mas de un atributo

    ul>li.item-$*10 -> esto genera una numeracion

    ul>li.item-$@101*10 -> esto define desde donde comienza una numeracion
